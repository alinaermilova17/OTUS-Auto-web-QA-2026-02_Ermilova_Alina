# Калькулятор среднего арифметического значения

Простая программа на Python для вычисления среднего арифметического чисел из списка.

## Описание

Проект демонстрирует базовые возможности Python:
- Создание функции для вычисления среднего арифметического
- Работа со списками чисел
- Вывод результатов в консоль

## Установка и запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/OtusTeam/QA-Python/blob/master/linter/fix_me.py
cd fix_my
```

2. Запустите программу:
```bash
python fix_me.py
```
